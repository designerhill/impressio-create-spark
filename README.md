# Impressio - AI-Powered Design Platform

A modern, responsive web application for creating stunning certificates, cards, and announcements with AI-powered design tools.

## 🎨 Features

### Core Functionality
- **Beautiful Certificates**: Professional certificate templates for achievements, courses, and recognition
- **Birthday Cards**: Personalized birthday cards with stunning designs and custom messages
- **Work Anniversaries**: Elegant work anniversary announcements and appreciation cards
- **Image Optimization**: Advanced tools to compress, resize, and enhance images without losing quality
- **AI-Powered Design**: Smart suggestions and automated layouts that adapt to your content
- **Custom Branding**: Add logos, colors, and fonts for consistent branded materials
- **Easy Sharing**: Share designs instantly with team members, clients, or on social media
- **Secure & Private**: Enterprise-grade security and privacy controls

### Design System
- **Modern Typography**: Enhanced font visibility with bold typography throughout
- **Responsive Design**: Optimized for all screen sizes and devices
- **Interactive Elements**: Smooth animations and hover effects
- **Accessibility**: WCAG compliant design with proper contrast ratios
- **Performance**: Fast loading times with optimized assets

### Pages & Navigation
- **Homepage**: Hero section with compelling CTAs and feature showcase
- **Features Page**: Detailed feature breakdown with statistics and examples
- **Templates Page**: Searchable template gallery with filtering capabilities
- **Pricing Page**: Transparent pricing plans with feature comparisons
- **About Page**: Company information and team details
- **Contact Page**: Customer support and inquiry forms

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd impressio-create-spark

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
# Build the application
npm run build

# Preview the production build
npm run preview
```

## 🛠️ Technology Stack

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite for fast development and optimized builds
- **Styling**: Tailwind CSS with custom design system
- **UI Components**: shadcn/ui for consistent, accessible components
- **Routing**: React Router for seamless navigation
- **Icons**: Lucide React for beautiful, consistent icons
- **State Management**: React Query for server state management
- **Animations**: CSS transitions and Framer Motion for smooth interactions
